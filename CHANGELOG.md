# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-03-14

### Added
- **File Organizer**
  - Automatic file classification and organization
  - Support for multiple file types (images, videos, audios, documents, archives)
  - Intelligent sub-category detection
  - Move or copy modes
  - Dry-run preview mode
  - Detailed logging and progress reporting

- **Data Cleaner**
  - Automatic data cleaning and standardization
  - Support for multiple formats (CSV, JSON, XML, TSV)
  - Automatic format detection
  - Data structure validation
  - Encoding and line ending normalization
  - Automatic backup creation

- **Documentation**
  - Comprehensive README
  - Usage guides and examples
  - API documentation
  - Installation instructions

### Technical
- Modular, extensible architecture
- Command-line interface
- Error handling and recovery
- Configuration system
