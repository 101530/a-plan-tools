### 长文案（2分钟）

"在数字时代，我们每天都会产生大量文件和数据。如何高效管理它们？A-Plan Tools 提供了两个强大的工具来解决你的问题。

**File Organizer - 文件整理工具**

你是否厌倦了混乱的文件目录？File Organizer 可以自动识别文件类型，将文件移动到对应的分类目录中。支持 7 大类别：图片、视频、音频、文档、压缩包、安装程序和其他文件。每个类别都有详细的子分类，如 wallpapers、screenshots、movies、tv-shows 等。

使用方法非常简单：
```bash
node file-organizer.js ~/Downloads
```

或者使用预览模式查看将要执行的操作：
```bash
node file-organizer.js ~/Downloads --dry-run
```

**Data Cleaner - 数据清洗工具**

数据文件往往包含各种格式和问题。Data Cleaner 可以自动清洗和标准化各种格式的数据文件。支持 CSV、JSON、XML、TSV 等格式。

功能包括：
- 自动格式检测
- 数据结构验证
- 编码标准化
- 换行符统一
- 空行移除
- 自动备份

使用方法：
```bash
node data-cleaner.js data.csv
node data-cleaner.js data.json --verbose
```
