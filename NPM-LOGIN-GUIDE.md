# npm 登录注册指南

## 📋 当前状态

❌ **未登录 npm**

## 🔐 执行登录注册

### 步骤 1：执行登录命令

```bash
npm adduser
```

### 步骤 2：输入信息

按照提示输入：
1. **Username**（用户名）
2. **Password**（密码）
3. **Email**（邮箱）
4. **确认密码**

### 步骤 3：验证登录

```bash
npm whoami
```

**预期输出：** 你的 npm 用户名

## 💡 提示

- 如果已经注册过，直接输入用户名和密码即可
- 密码不会显示，输入后直接按回车
- 确保记住用户名和密码
