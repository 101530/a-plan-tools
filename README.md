# A-Plan Tools

A-Plan Tools 是一套实用的数字产品工具集，帮助用户提高工作效率，自动化文件整理和数据清洗流程。

## 📦 包说明

### file-organizer

自动整理文件到分类目录的 Node.js 工具。

**安装：**
```bash
npm install -g file-organizer
```

**使用：**
```bash
file-organizer /path/to/files
```

### data-cleaner

自动清洗和标准化数据文件的 Node.js 工具。

**安装：**
```bash
npm install -g data-cleaner
```

**使用：**
```bash
data-cleaner data.csv
```

## 🚀 快速开始

### 方法 1：全局安装

```bash
# 安装工具
npm install -g file-organizer data-cleaner

# 使用工具
file-organizer ~/Downloads
data-cleaner data.csv
```

### 方法 2：本地安装

```bash
# 安装依赖
npm install

# 使用工具
npx file-organizer ~/Downloads
npx data-cleaner data.csv
```
