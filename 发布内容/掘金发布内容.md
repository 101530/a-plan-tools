# 开源Node.js工具：File Organizer & Data Cleaner，提升开发效率

最近我开发了两个实用的Node.js CLI工具，帮助开发者自动化文件整理和数据清洗工作。

## 📁 File Organizer - 文件整理工具

### 功能介绍
自动按文件类型分类整理文件，支持以下分类：
- **图片** → images/
- **视频** → videos/
- **音频** → audios/
- **文档** → documents/
- **压缩包** → archives/

### 核心特点
✅ MIT开源协议
✅ 跨平台支持（Windows、Mac、Linux）
✅ 可自定义分类规则
✅ 自动创建备份
✅ 详细的操作报告

### 使用示例
```bash
npm install -g a-plan-tools-file-organizer

# 整理指定目录
organize ~/Downloads

# 预览模式（不实际移动文件）
organize ~/Downloads --dry-run
```

---

## 🧹 Data Cleaner - 数据清洗工具

### 功能介绍
快速清理和标准化各种格式的数据文件，支持：
- **CSV** - 逗号分隔值
- **JSON** - JavaScript对象
- **XML** - 可扩展标记语言
- **TSV** - 制表符分隔值

### 核心特点
✅ MIT开源协议
✅ 跨平台支持
✅ 自动检测文件格式
✅ 移除空行和空白字符
✅ 标准化换行符
✅ 自动创建备份
✅ 支持预览模式
✅ 详细统计报告

### 使用示例
```bash
npm install -g a-plan-tools-data-cleaner

# 清洗CSV文件
data-cleaner data.csv

# 清洗JSON文件
data-cleaner data.json

# 预览模式
data-cleaner data.csv --dry-run

# 输出到新文件
data-cleaner data.csv --output cleaned.csv
```

## 🚀 快速开始

### 安装
```bash
# 全局安装两个工具
npm install -g a-plan-tools-file-organizer a-plan-tools-data-cleaner

# 或使用npx直接运行
npx a-plan-tools-file-organizer ~/Downloads
npx a-plan-tools-data-cleaner data.csv
```
