### 帖子内容
```
分享两个最近开发的Node.js CLI工具：

## 1. File Organizer - 文件整理工具
自动按类型分类文件，节省大量手动整理时间。

**功能：**
- 图片 → images/
- 视频 → videos/
- 文档 → documents/
- 压缩包 → archives/

**特点：**
- MIT开源协议
- 跨平台支持
- 可自定义规则
- 自动创建备份

**npm包：** a-plan-tools-file-organizer
**GitHub：** https://github.com/101530/a-plan-tools

---

## 2. Data Cleaner - 数据清洗工具
快速清理和标准化各种格式数据文件。

**功能：**
- CSV、JSON、XML、TSV文件清洗
- 移除空行、空白字符
- 标准化换行符
- 自动备份

**特点：**
- MIT开源协议
- 跨平台支持
- 支持预览模式
- 详细统计报告

**npm包：** a-plan-tools-data-cleaner
**GitHub：** https://github.com/101530/a-plan-tools

---

两个工具都MIT开源，欢迎大家使用和贡献！
```
