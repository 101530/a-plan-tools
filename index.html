工具介绍

File Organizer - 文件整理工具

自动识别文件类型，将文件分类到对应的目录中。支持 7 大类别，每个类别都有详细的子分类。

  • 智能识别文件类型
  • 自动分类到 7 大类别
  • 支持移动或复制模式
  • 预览模式查看操作结果
  • 详细的操作日志

使用方法

# 整理 Downloads 目录
node file-organizer.js ~/Downloads

# 预览模式
node file-organizer.js ~/Downloads --dry-run

Data Cleaner - 数据清洗工具

自动清洗和标准化各种格式的数据文件。支持 CSV、JSON、XML、TSV 等格式。

  • 自动格式检测
  • 支持 CSV、JSON、XML、TSV
  • 数据结构验证
  • 编码和换行符标准化
  • 自动备份机制

使用方法

# 清洗 CSV 文件
node data-cleaner.js data.csv

# 清洗 JSON 文件
node data-cleaner.js data.json --verbose
