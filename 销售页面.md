# A计划 - 数字产品销售页面

## 🎉 欢迎使用

两个实用的命令行工具，帮助你提高工作效率。

## 📦 产品1：文件整理工具 ⭐

**价格**: 免费
**评分**: ⭐⭐⭐⭐⭐

### 核心功能
- 📁 **自动识别**：智能识别15+种文件类型
- 🎯 **一键整理**：自动分类整理文件
- 🚀 **快速执行**：秒级处理大量文件
- 🔍 **预览模式**：安全预览，避免误操作

### 使用场景
- 整理下载文件夹
- 分类照片和文档

### 如何使用
```bash
# 安装
npm install -g file-organizer

# 整理当前目录
organize

# 预览（不实际移动文件）
organize --dry-run
```

## 🧹 产品2：数据清洗工具 ⭐

**价格**: 免费
**评分**: ⭐⭐⭐⭐⭐

### 核心功能
- 🧹 **数据清洗**：快速清理JSON数据
- 🗑️ **智能过滤**：移除空字符串和null值
- 📁 **支持嵌套**：处理复杂的JSON结构
- 🚀 **自动格式化**：输出格式化的JSON

### 如何使用
```bash
# 基本用法
node data-cleaner.js input.json output.json

# 移除空字符串和null值
node data-cleaner.js input.json output.json --remove-empty --remove-null
```

---

## 🚀 立即开始

### 步骤1：安装
```bash
npm install -g file-organizer
npm install -g data-cleaner
```

### 步骤2：使用
```bash
# 整理文件
organize

# 清洗数据
node data-cleaner.js input.json output.json --remove-empty --remove-null
```
